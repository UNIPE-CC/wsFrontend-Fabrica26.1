# wsFrontend-Fabrica26.1
Projeto frontend utilizando API.

Este é um projeto desenvolvido com [Next.js](https://nextjs.org) criado através do [`create-next-app`](https://nextjs.org/docs/app/api-reference/cli/create-next-app).

- npx create-next-app@latest

---

## Como iniciar o projeto

Primeiro:
- npm i
Atualiza o npm

Execute o servidor de desenvolvimento:

```bash
npm run dev
# ou
yarn dev
# ou
pnpm dev
# ou
bun dev
```
clique no Ctrl + botão do mouse na URL que aparece no terminal, assim você verá o projeto no seu navegador.

ou

Abra o navegador e digite na barra de pesquisa localhost:3000 (http://localhost:3000) o projeto ira rodar no navegador.
